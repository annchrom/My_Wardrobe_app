# My_Wardobe_app
My Wardrobe is a web application for managing a personal wardrobe.
=======
MyWardrobe is a web application for managing a personal wardrobe.

The application allows users to store clothing items with photos, create outfits, track clothing usage, and analyze wardrobe statistics.

## Project Goals

The main goal of the application is to help users:

- Organize their wardrobe
- Store clothing photos
- Create outfit combinations
- Track how often clothes are worn
- Rate outfits
- Analyze wardrobe usage

## Planned Features

### User Management

- User registration
- User login
- Personal wardrobe for each user

### Clothing Management

- Add clothing items
- Upload clothing photos
- Edit clothing details
- Delete clothing items

### Clothing Attributes

- Name
- Category
- Color
- Season
- Photo
- Wear count

### Outfit Management

- Create outfits
- Add clothing items to outfits
- Rate outfits
- Track outfit usage

### Filters

- Filter by category
- Filter by color
- Filter by season

### Statistics

- Most worn clothing items
- Least worn clothing items
- Outfit ratings
- Total wardrobe size
